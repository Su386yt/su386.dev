SySi

Link: https://github.com/Su386yt/sysi SySi is a static site generator, compiles .syl, .md, .html, and regular .txt files to .html and inserts them into a designated template. It supports macros, called "fragments" and multiple different templates. Currently .syl files are the most limited and currently do not support ordered lists. I also plan to support photo galleries, GitHub sync (automatically taking projects from the user's GitHub profile, and displaying it on their website), and runnable scripts. For more information visit the project page or the "how to" page. To see this website's source code, visit the site's GitHub Repo.
